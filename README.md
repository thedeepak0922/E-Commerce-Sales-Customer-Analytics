# E-Commerce Sales & Customer Analytics

## Project Overview
This project focuses on data cleaning and preprocessing of an online retail dataset using Python and Pandas.

## Objectives
- Handle missing values
- Remove cancelled orders
- Remove invalid quantities and prices
- Create TotalPrice column
- Perform date-time feature extraction
- Export cleaned dataset

## Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Files Included
- E-Commerce Sales & Customer Analytics.ipynb
- OnlineRetail_clean.xls

## Dataset
Online Retail Dataset (UCI ML Repository)

## Outcome
Prepared a clean and analysis-ready retail dataset for further analytics and customer segmentation.
